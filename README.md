
# Personalized E-Commerce Recommendation System using High-Utility Sequential Mining
##  CS691-Spring2025-Team8-Capstone Project

## **Project Description**  
This project implements a **personalized recommendation system** for e-commerce platforms using **High-Utility Sequential Mining (HUIM)**. It considers multiple factors such as customer preferences, purchase history, item popularity, and price sensitivity to generate personalized product recommendations.

## **Features**  
-  Personalized product recommendations  
-  High-Utility Itemset Mining (HUIM) for better suggestions  
-  Sequential pattern analysis to capture time-based purchasing trends  
-  User-friendly interface built with **React.js and Tailwind CSS**  
-  Backend API using **Flask and MySQL**  

## **Tech Stack**  
- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Flask, Python, MySQL, Firebase  
- **Machine Learning:** Pandas, NumPy, Scikit-learn, HUIM  
- **Visualization:** Matplotlib, Seaborn, Plotly  

## **How It Works**  
1. The system analyzes user purchase history and product interactions.  
2. It applies **High-Utility Sequential Mining (HUIM)** to identify buying patterns.  
3. The recommendation engine suggests products based on past purchases, item utility, and sequential trends.
