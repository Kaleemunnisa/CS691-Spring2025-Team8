# CS691-Spring2025-Team8

## 📌 Problem Statement
> This project aims to enhance e-commerce product recommendation systems by utilizing **High-Utility Itemset Mining (HUIM)** with varying utility thresholds. Unlike traditional methods, this system will consider multiple factors such as **customer preferences, purchase history, item popularity, and price sensitivity** to generate **personalized product recommendations**. 

## 📌 Hypothesis
By integrating **HUIM with multiple thresholds**, we hypothesize that:
1. Personalized recommendations will improve user experience and increase purchase rates.
2. Factoring in **time-based sequential patterns** will enhance recommendation accuracy.
3. The proposed system will outperform traditional models in predicting user preferences.
